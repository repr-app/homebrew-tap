# Homebrew Tap for repr

This is the official [Homebrew](https://brew.sh) tap for [repr](https://repr.dev) - a privacy-first CLI that analyzes your git repositories and generates developer profiles.

## Installation

```bash
brew tap repr-app/tap
brew install repr
```

## Updating

```bash
brew upgrade repr
```

## Uninstalling

```bash
brew uninstall repr
brew untap repr-app/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `repr` | Privacy-first CLI for developer profiles |
